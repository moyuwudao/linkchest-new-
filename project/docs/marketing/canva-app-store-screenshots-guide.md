# LinkChest 应用商店截图 — Canva 设计规范

> 本规范用于在 Canva 中统一制作 iOS / Android 应用商店上架截图。
> 建议先通读全文，再逐张制作。

---

## 一、品牌基础

### 1.1 品牌色

| 用途 | 色值 | 说明 |
|------|------|------|
| 主色 | `#1B2A4A` | 深蓝，用于标题文字、按钮、重点图形 |
| 强调色 | `#C8956C` | 暖金，用于小标签、装饰点、CTA 高亮 |
| 背景色 | `#F7F5F0` | 米白，截图主背景 |
| 卡片白 | `#FFFFFF` | 手机样机外框、信息卡片 |
| 主文字 | `#2D3142` | 近黑，用于正文说明 |
| 次要文字 | `#5A5A5A` | 灰色，用于辅助说明 |

### 1.2 字体建议

- **中文标题**：思源黑体 / 阿里巴巴普惠体 Bold
- **中文说明**：思源黑体 / 阿里巴巴普惠体 Regular
- **英文/数字**：Montserrat / Inter / Poppins
- 字重：标题 Bold（700），说明 Regular（400）或 Medium（500）

### 1.3 Logo 使用

- 文件位置：`project/assets/master/icon-master.png` 或 `project/assets/linkchest.svg`
- 每张图右上角或左上角固定放置 Logo，高度 48-64px
- 保持原比例，不要拉伸

## 二、画布尺寸

### 2.1 iOS App Store

| 尺寸 | 用途 |
|------|------|
| **1242 × 2688 px** | iPhone 6.7 英寸（主要尺寸，优先做） |
| 1290 × 2796 px | iPhone 6.7 英寸（备选） |
| 1170 × 2532 px | iPhone 6.1 英寸 |
| 1179 × 2556 px | iPhone 14 Pro / 15 / 16 |

### 2.2 Google Play / 国内安卓商店

| 尺寸 | 用途 |
|------|------|
| **1080 × 2160 px** | 通用安卓 18:9 |
| 1080 × 1920 px | 通用安卓 16:9 |

### 2.3 Canva 操作

1. 点击 **"创建设计" → "自定义大小"**
2. 输入 **1242 × 2688 px**（iOS 主尺寸）
3. 第一张做完后，点击 **"复制页面"** 保持版式统一
4. 最后导出 PNG，再用 Canva 的 **"调整大小"** 生成其他尺寸（Pro 功能）

## 三、版式规范

### 3.1 通用布局

```
┌─────────────────────────────┐
│  [Logo]                     │  ← 顶部边距 80px
│                             │
│  大标题（一句话利益点）        │  ← 左侧边距 100px
│  小字说明（不超过 14 字）      │
│                             │
│         [手机样机]            │  ← 居中偏右，宽度 55%-65%
│                             │
└─────────────────────────────┘
```

### 3.2 安全边距

- 上下左右各留 **80-100px**
- 文字不要贴边，避免被商店裁剪
- 手机样机底部距离下边缘 **120px 以上**

### 3.3 字号规范（以 1242px 宽度画布为基准）

| 元素 | 字号 | 字重 | 行高 |
|------|------|------|------|
| 大标题 | 72-84px | Bold | 1.2 |
| 小字说明 | 36-42px | Regular/Medium | 1.4 |
| 标签/角标 | 28-32px | Medium | 1.2 |

### 3.4 手机样机

- 在 Canva **"元素"** 中搜索 `iPhone mockup` 或 `Android mockup`
- 选择带透明屏幕的样机，或用 **"框架（Frame）"** 功能直接拖入截图
- 样机宽度占画布宽度的 **55%-65%**
- 样机角度：5 张图保持一致，建议正面微侧或纯正面

## 四、5 张截图内容规范

### 第 1 张：价值主张

- **目的**：3 秒内让用户知道这是什么 App
- **大标题**：收藏一切链接
- **小字说明**：复制链接，自动保存标题与封面
- **手机截图场景**：
  - 新建收藏页，输入框中粘贴一条链接
  - 系统已自动抓取标题和封面
- **视觉建议**：
  - 背景用主色 `#1B2A4A` 渐变到深蓝
  - 文字用白色
  - 手机样机下方可加微弱金色光晕 `#C8956C`

### 第 2 张：分组与标签管理

- **目的**：展示整理能力
- **大标题**：分组 + 标签双维度
- **小字说明**：再多收藏也能 3 秒找到
- **手机截图场景**：
  - 收藏列表页，顶部有分组标签栏
  - 列表中显示若干带封面、标题、标签的收藏卡片
- **视觉建议**：
  - 背景用米白 `#F7F5F0`
  - 可在手机样机旁漂浮 1-2 个标签图形（如 `#旅行` `#教程`）

### 第 3 张：一键导入书签

- **目的**：展示迁移能力，降低换 App 门槛
- **大标题**：一键导入浏览器书签
- **小字说明**：历史收藏不丢失
- **手机截图场景**：
  - 导入页面，显示导入成功结果
  - 或 HTML 书签文件导入后的列表页
- **视觉建议**：
  - 背景用浅金色 `#C8956C` 10% 透明度
  - 可在角落加一个 "Import" 标签

### 第 4 张：生成分享页面

- **目的**：展示社交传播能力
- **大标题**：精选收藏生成分享页
- **小字说明**：好友无需安装即可查看
- **手机截图场景**：
  - 创建分享页，预览分享卡片
  - 或分享详情页，显示一组精选收藏
- **视觉建议**：
  - 手机样机右侧可加一个 "分享" 图标或链接图形
  - 强调色 `#C8956C` 用于分享按钮

### 第 5 张：三端实时同步

- **目的**：展示多端生态
- **大标题**：手机、网页、插件三端同步
- **小字说明**：随时随地访问你的收藏
- **手机截图场景**：
  - 手机收藏列表 + 小的 Chrome 扩展 / Web 界面示意
- **视觉建议**：
  - 主手机样机 + 两个较小的样机/界面缩略图
  - 用细线和同步箭头暗示数据同步

## 五、截图素材准备清单

在制作 Canva 之前，先用测试账号准备以下内容：

- [ ] 登录 App，使用美观的测试数据（10-20 条真实链接）
- [ ] 推荐的示例链接类型：
  - 小红书笔记
  - B 站视频
  - YouTube 视频
  - 微信公众号文章
  - 优质博客/教程
- [ ] 创建 3-5 个分组：如 "设计灵感"、"教程"、 "待看"、"旅行"
- [ ] 添加 5-10 个标签
- [ ] 创建 1 个分享页面
- [ ] 确保截图中不出现：
  - 低电量提示
  - 通知弹窗
  - 测试账号昵称（可用 "LinkChest User"）
  - 空状态、错误提示

## 六、Canva 制作流程

### 步骤 1：创建画布

1. 打开 Canva
2. 点击 **"创建设计" → "自定义大小"**
3. 输入 **1242 × 2688 px**
4. 命名：`LinkChest App Store Screenshots`

### 步骤 2：建立母版

1. 设置背景色 `#F7F5F0`
2. 上传 Logo，放到左上角，高度 56px
3. 添加文字占位：
   - 大标题：思源黑体 Bold，78px，颜色 `#1B2A4A`
   - 小字：思源黑体 Regular，38px，颜色 `#5A5A5A`
4. 添加手机样机占位（可先不放截图）

### 步骤 3：制作第 1 张

1. 复制母版画布
2. 替换标题和说明
3. 上传真机截图，拖入手机样机
4. 调整细节

### 步骤 4：批量复制

1. 右键点击画布 → **"复制页面"**
2. 依次制作 2-5 张
3. 保持样机大小、位置、Logo 位置一致

### 步骤 5：导出

1. 点击 **"分享" → "下载"**
2. 文件类型：**PNG**
3. 勾选需要导出的全部 5 张页面
4. 点击下载

### 步骤 6：适配多尺寸

1. 复制整个设计
2. 使用 **"调整大小"** 功能改为 1080×2160（Canva Pro）
3. 微调文字和样机大小
4. 重新导出

## 七、质量检查清单

每张图导出前检查：

- [ ] 标题文字在缩略图尺寸下仍可辨认
- [ ] 手机截图清晰，无模糊或拉伸
- [ ] 5 张图风格统一（字体、颜色、间距、样机角度）
- [ ] 没有截到系统状态栏的私人信息
- [ ] 没有空列表、错误提示、测试数据
- [ ] Logo 位置一致，未被裁剪
- [ ] 背景与手机样机有足够对比度

## 八、常见平台特殊要求

### iOS App Store

- 必须提供 6.7 英寸 iPhone 截图
- 截图不能包含 iOS 状态栏（部分审核会卡）
- 避免使用 "Free"、"Best" 等绝对化宣传词

### Google Play

- 最多 8 张截图
- 截图不能包含透明背景
- 建议提供手机、平板两种尺寸

### 国内安卓商店（华为/小米/OPPO/vivo）

- 通常要求 3-5 张竖屏截图
- 截图中不要出现竞品 Logo
- 部分商店要求首图不能是引导页或 Splash
