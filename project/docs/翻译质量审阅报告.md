## 五、审阅重点

请审阅 AI 重点检查以下问题：

1. **语气与正式程度**
   - FR：法语登录表单是否使用合适的正式语气（vous）？
   - DE：德语是否正确区分 Siezen（您）和 Duzen（你）？

2. **文化适配**
   - JA：`私人藏馆`、`宝库` 等词的日语表达是否自然，还是过于生硬直译？
   - KO：`私家珍藏`、`宝藏` 等词韩语是否口语化？

3. **占位符完整性**
   - 所有 `{placeholder}` 是否原样保留？特别检查含多个占位符的句子（如 `importConfirmDesc`）

4. **引号格式**
   - FR：`「{title}」` / DE：`„{title}"` / KO：`«{title}»` 在各语言 UI 中是否合适？
   -JA：`「{title}」` 是否符合日语习惯？

5. **品牌词检查**
   - `LinkChest` 所有语言均未翻译，是否符合预期？
   - `TikTok` / `YouTube` / `Amazon` 等平台名是否正确保留？
