| 项目 | 值 |

## 三、上传发布

### 3.1 登录vivo开放平台

1. 访问 https://dev.vivo.com.cn
2. 使用已注册的开发者账号登录
3. 进入「应用分发」→「应用管理」→「新增应用」

### 3.2 填写应用信息

| 字段 | 填写内容 | 备注 |
|------|---------|------|
| 应用名称 | 链藏 | ≤15字 |
| 包名 | com.linkchest.app | - |
| 版本号 | 构建时的 versionCode | - |
| 分类 | 效率工具 | - |
| 短描述 | 你的数字收藏宝库 | ≤10字 |
| 长描述 | 见产品基础信息详细介绍 | ≤1000字 |
| 关键词 | 收藏,链接管理,数字资产,笔记,书签,整理,分享,备份,同步 | 逗号分隔 |
| 隐私政策URL | https://linkchest.cn/privacy | 必须可访问 |
| 客服邮箱 | support@linkchest.cn | - |

### 3.3 上传材料

| 材料 | 规格 | 数量 |
|------|------|------|
| 应用图标 | 512x512 PNG | 1张 |
| 应用截图 | 1080x1920 或 1080x2400 PNG/JPEG | 3-5张 |
| APK文件 | 已签名Release APK | 1个 |

### 3.4 上传软著证书

- 格式：PDF扫描件
- 路径：后台「资质管理」→「软件著作权」

## 五、注意事项

### 5.1 vivo特殊要求

| 要求 | 说明 |
|------|------|
| APK加固 | vivo推荐但不强制加固，加固后需测试兼容性 |
| 64位支持 | 建议同时支持32位和64位架构 |
| 隐私合规检测 | vivo有自动化检测，确保首次启动有隐私政策弹窗 |

### 5.2 常见被拒原因

| 原因 | 解决方案 |
|------|---------|
| 未提供隐私政策同意弹窗 | 首次启动弹出隐私政策并需用户同意 |
| APK含不必要权限 | 移除 build.gradle 中未使用的权限 |
| 应用名称已被占用 | 确认「链藏」名称可用 |
| 截图模糊或尺寸不对 | 使用标准尺寸重新截图 |

### 5.3 审核周期

| 阶段 | 预计时间 |
|------|---------|
| 首次提交审核 | 2-5个工作日 |
| 版本更新审核 | 1-3个工作日 |

### 5.4 隐私合规特别注意

vivo 对隐私合规要求较为严格，确保：
1. 首次启动时弹出隐私政策弹窗
2. 用户同意前不收集任何个人信息
3. 权限申请在用户使用时才弹出系统授权对话框
4. 隐私政策中包含数据收集、使用、共享说明
