### 2.3 详细介绍

#### 国内版

```
链藏是一款专为数字内容爱好者打造的收藏管理工具。无论是视频、文章、图片还是社交平台内容，都能一键收藏、智能归类。

核心亮点：
• 自动封面获取 - 添加链接自动提取精美封面，收藏即展示
• 多端实时同步 - Android、iOS、Web、Chrome插件无缝同步，随时随地管理收藏
• 智能分类管理 - 自定义标签和分类，快速定位收藏内容
• 数据备份无忧 - 支持本地导出和云端备份，数据安全可控
• 自定义展示风格 - 自定义排序规则、展示布局，打造专属收藏主页
• 多语言支持 - 中文、英文自由切换，国际化体验

支持YouTube、B站、小红书、抖音、Twitter等主流平台链接收藏。免费版即可体验核心功能，付费版解锁更多高级特性。
```

#### 海外版

```
LinkChest is a collection management tool designed for digital content enthusiasts. Whether it's videos, articles, images, or social media content, save it all in one place.

Key Features:
• Auto Cover Extraction - Beautiful covers are automatically fetched when you save a link
• Cross-Platform Sync - Seamlessly sync across Android, iOS, Web, and Chrome Extension
• Smart Organization - Custom tags and categories to quickly find your collections
• Data Backup - Local export and cloud backup options keep your data safe
• Personalized Display - Customize sort orders and display layouts for your collection page
• Multi-Language Support - Switch between English and Chinese effortlessly

Supports collection from YouTube, Twitter, Instagram, and more popular platforms. Start free and upgrade to unlock premium features.
```
