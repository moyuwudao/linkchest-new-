### 1.4 软件用途

LinkChest 链藏是一款跨平台的收藏聚合管理工具，主要功能包括：

1. **内容收藏**：支持抖音、小红书、淘宝、微博、哔哩哔哩、知乎等主流平台内容的收藏
2. **智能分类**：自动识别链接所属平台，支持自定义标签和列表管理
3. **跨平台同步**：Web端、移动端数据实时同步
4. **内容分享**：支持生成分享链接，方便与他人共享收藏内容
5. **封面生成**：自动为收藏内容生成封面图片
6. **搜索筛选**：支持按关键词、标签、平台进行筛选和搜索

## 二、技术特点

### 2.1 技术架构

**架构模式：** 前后端分离，微服务架构

**技术栈：**

| 层级 | 技术 | 说明 |
|------|------|------|
| 前端（Web） | Next.js 14 | React框架，App Router模式 |
| 前端（Mobile） | React Native + Expo | 跨平台移动应用开发 |
| 后端 | Node.js + Express | 服务器端API开发 |
| 数据库 | PostgreSQL | 关系型数据库 |
| ORM | Prisma | 数据库访问层 |
| 状态管理 | Zustand (Mobile) | 轻量级状态管理 |
| 数据获取 | React Query | 服务端状态管理 |
| UI框架 | Tailwind CSS (Web) | CSS框架 |
| 图标库 | Lucide React | 图标组件库 |

### 2.2 核心技术特点

#### 2.2.1 跨平台架构

- **Web端**：采用 Next.js 14 App Router，支持服务端渲染和客户端渲染
- **移动端**：采用 React Native + Expo，一套代码同时支持iOS和Android
- **数据同步**：通过RESTful API实现多端数据实时同步

#### 2.2.2 智能平台识别

- 自动解析URL，识别链接所属平台
- 支持抖音、小红书、淘宝、微博、哔哩哔哩、知乎等主流平台
- 根据平台类型自动匹配图标和颜色主题

#### 2.2.3 封面生成技术

- 自动抓取网页内容生成封面图片
- 支持多种封面模板样式
- 支持自定义封面上传

#### 2.2.4 安全机制

- JWT Token认证机制
- 密码加密存储（bcryptjs）
- HTTPS加密传输
- 安全的API请求拦截

#### 2.2.5 国际化支持

- 支持6种语言：中文、English、日本語、한국어、Français、Deutsch
- 基于i18next的国际化框架
- 支持语言切换和自动检测

### 2.3 功能模块划分

| 模块 | 描述 | 位置 |
|------|------|------|
| 用户认证 | 登录、注册、Token管理 | auth |
| 收藏管理 | 添加、编辑、删除收藏 | collections |
| 列表管理 | 创建、编辑、删除列表 | lists |
| 标签管理 | 创建、编辑、删除标签 | tags |
| 分享功能 | 生成分享链接、权限管理 | shares |
| 数据统计 | 用户数据统计 | stats |
| 国际化 | 多语言支持 | i18n |

### 2.4 数据结构

**用户（User）：**
- id: 用户唯一标识
- email: 邮箱地址
- username: 用户名（可选）
- nickname: 昵称（可选）
- avatar: 头像（可选）
- createdAt: 创建时间
- updatedAt: 更新时间

**收藏（Collection）：**
- id: 收藏唯一标识
- url: 链接地址
- title: 标题
- coverImage: 封面图片（可选）
- platform: 平台类型
- note: 备注（可选）
- rating: 评分（可选）
- tags: 标签列表
- lists: 所属列表
- createdAt: 创建时间
- updatedAt: 更新时间

**列表（List）：**
- id: 列表唯一标识
- name: 列表名称
- parentId: 父列表ID（可选）
- isDefault: 是否默认列表
- collectionCount: 收藏数量

**标签（Tag）：**
- id: 标签唯一标识
- name: 标签名称
- collectionCount: 关联收藏数量

**分享（Share）：**
- id: 分享唯一标识
- title: 分享标题
- description: 描述（可选）
- collectionIds: 收藏ID列表
- isPublic: 是否公开
- password: 访问密码（可选）
- viewCount: 访问次数
- createdAt: 创建时间
- expiresAt: 过期时间（可选）

## 三、软件功能介绍

### 3.1 核心功能

#### 3.1.1 收藏管理
- 添加新收藏：支持粘贴链接，自动识别平台
- 编辑收藏：修改标题、封面、标签等信息
- 删除收藏：支持单条删除和批量删除
- 查看收藏：网格视图和列表视图切换

#### 3.1.2 分类管理
- 列表管理：创建、编辑、删除收藏列表
- 标签管理：创建、编辑、删除标签
- 批量操作：批量添加标签和列表

#### 3.1.3 搜索筛选
- 关键词搜索：支持标题搜索
- 标签筛选：按标签筛选收藏
- 平台筛选：按平台类型筛选

#### 3.1.4 分享功能
- 创建分享：选择收藏生成分享链接
- 权限设置：公开分享和私有分享
- 密码保护：为私有分享设置访问密码
- 访问统计：记录分享访问次数

#### 3.1.5 用户中心
- 用户信息：显示邮箱、注册日期
- 数据统计：收藏、列表、标签数量
- 主题切换：深色/浅色模式切换
- 设置管理：语言、通知等设置

### 3.2 辅助功能

- 自动封面生成
- 收藏评分系统
- 备注功能
- 撤销操作支持
- Toast通知提示
