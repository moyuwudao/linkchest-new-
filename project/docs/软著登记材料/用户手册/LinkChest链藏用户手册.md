## 1. 软件概述

### 1.1 软件简介

LinkChest 链藏是一款跨平台的收藏聚合管理工具，旨在帮助用户高效地收集、整理和管理全网优质内容。

### 1.2 核心价值

- **一站式收藏**：支持抖音、小红书、淘宝、微博、哔哩哔哩、知乎等主流平台内容收藏
- **智能分类**：自动识别平台类型，支持自定义标签和列表管理
- **跨平台同步**：Web端、移动端数据实时同步
- **内容分享**：支持生成分享链接，方便与他人共享收藏

### 1.3 支持平台

| 平台 | 图标 | 说明 |
|------|------|------|
| 抖音 | 📹 | 短视频内容收藏 |
| 小红书 | 📕 | 图文笔记收藏 |
| 淘宝 | 🛒 | 商品链接收藏 |
| 微博 | 📢 | 微博内容收藏 |
| 哔哩哔哩 | 📺 | 视频内容收藏 |
| 知乎 | 💡 | 问答内容收藏 |

## 4. 用户注册与登录

### 4.1 注册流程

1. 打开应用，点击"注册"按钮
2. 输入邮箱地址
3. 设置登录密码（至少6位）
4. 确认密码
5. 点击"注册"完成注册
6. 返回登录页面进行登录

### 4.2 登录流程

1. 打开应用，进入登录页面
2. 输入注册邮箱
3. 输入密码
4. 点击"登录"按钮
5. 登录成功后进入收藏列表页面

### 4.3 密码要求

- 密码长度至少6位
- 支持字母、数字、特殊字符
- 区分大小写

### 4.4 账号安全

- 登录失败次数限制，连续5次失败后账号锁定30分钟
- 支持JWT Token认证，Token有效期7天
- 支持自动登录功能

### 4.5 退出登录

1. 进入个人中心页面
2. 点击"退出登录"按钮
3. 确认退出操作
4. 退出后清除本地登录状态

## 5. 收藏管理

### 5.1 添加收藏

**操作步骤：**

1. 点击页面右下角"+"按钮
2. 在输入框中粘贴或输入链接地址
3. 系统自动识别平台类型
4. 输入收藏标题（可自动获取）
5. 点击"生成封面"生成内容封面
6. 选择收藏列表（可多选）
7. 添加标签（可选）
8. 对内容进行评分（可选）
9. 添加备注信息（可选）
10. 点击"保存收藏"完成添加

### 5.2 查看收藏

**列表视图：**
- 显示收藏的缩略图、标题、平台标识
- 支持网格视图和列表视图切换
- 点击收藏项可查看详情

**筛选功能：**
- 按标签筛选
- 按平台筛选
- 关键词搜索

### 5.3 编辑收藏

1. 在收藏列表中找到要编辑的收藏
2. 点击进入详情页面
3. 点击"编辑"按钮
4. 修改标题、封面、标签、评分等信息
5. 点击"保存"完成编辑

### 5.4 删除收藏

**单条删除：**
1. 在收藏列表中找到要删除的收藏
2. 长按或点击删除按钮
3. 确认删除操作

**批量删除：**
1. 在收藏列表中勾选多个收藏
2. 点击底部"删除"按钮
3. 确认批量删除操作

## 6. 标签与分类

### 6.1 列表管理

**创建列表：**
1. 进入"管理"页面
2. 切换到"列表"标签
3. 在输入框中输入列表名称
4. 按回车键或点击"+"按钮创建

**编辑列表：**
1. 在列表列表中找到要编辑的列表
2. 点击编辑图标
3. 修改列表名称
4. 按回车键确认

**删除列表：**
1. 在列表列表中找到要删除的列表
2. 点击删除图标
3. 确认删除操作

### 6.2 标签管理

**创建标签：**
1. 进入"管理"页面
2. 切换到"标签"标签
3. 在输入框中输入标签名称
4. 按回车键或点击"+"按钮创建

**编辑标签：**
1. 在标签列表中找到要编辑的标签
2. 点击编辑图标
3. 修改标签名称
4. 按回车键确认

**删除标签：**
1. 在标签列表中找到要删除的标签
2. 点击删除图标
3. 确认删除操作

### 6.3 智能封面生成

**自动生成封面：**
1. 添加收藏时，系统自动尝试从目标网页抓取封面图
2. 支持多种封面模板，根据平台自动适配
3. 点击"重新生成"可更换封面样式

**手动设置封面：**
1. 在收藏详情或编辑页面，点击封面区域
2. 可选择上传本地图片作为封面
3. 支持自动裁剪和缩放适配

### 6.4 收藏导入

**支持导入格式：**
- Netscape Bookmark HTML（浏览器书签导出格式）
- JSON格式（LinkChest备份文件）

**操作步骤：**
1. 进入"数据管理"或"导入导出"页面
2. 点击"导入收藏"按钮
3. 选择导入文件（.html 或 .json）
4. 系统自动解析并导入收藏内容
5. 显示导入结果（成功数/失败数）

### 6.5 收藏导出

**支持导出格式：**
- Netscape Bookmark HTML（可直接导入浏览器）
- JSON（可用于数据备份和迁移）
- CSV（可在Excel等表格软件中查看）

**操作步骤：**
1. 进入"导出"页面
2. 选择导出格式
3. 点击导出按钮
4. 系统生成下载文件并自动下载

### 6.6 自动备份

**功能说明：**
自动备份功能定期将收藏数据导出为JSON文件保存到用户指定位置。

**备份频率设置：**
- 每天一次
- 每周一次
- 每月一次
- 手动备份

**使用步骤：**
1. 进入"自动备份"页面
2. 选择备份频率
3. 点击"立即备份"可手动触发备份
4. 备份文件自动保存

### 6.7 重复收藏检测

**功能说明：**
系统自动检测收藏列表中的重复链接，帮助用户清理冗余数据。

**使用步骤：**
1. 进入"重复检测"页面
2. 系统自动扫描并列出重复的收藏
3. 选择要合并或删除的重复项
4. 点击"合并"保留一条记录
5. 或点击"删除"移除多余的收藏

### 6.8 平台统计

**功能说明：**
通过图表形式展示收藏在不同平台的分布情况。

**查看方式：**
1. 进入"平台统计"页面
2. 查看各平台收藏数量柱状图
3. 查看平台占比百分比
4. 点击平台可查看该平台下的收藏列表

## 7. 分享功能

### 7.1 创建分享

1. 进入分享页面
2. 输入分享标题
3. 添加分享描述（可选）
4. 选择要分享的收藏
5. 设置分享权限（公开/私有）
6. 设置访问密码（可选）
7. 点击"生成分享链接"

### 7.2 查看分享

1. 通过分享链接访问分享页面
2. 如果是私有分享，输入访问密码
3. 查看分享的收藏列表
4. 点击收藏可查看详细信息

### 7.3 分享设置

- **公开分享**：任何人都可以通过链接访问
- **私有分享**：需要密码才能访问
- **访问统计**：显示分享的访问次数
- **过期时间**：支持设置1小时、24小时、7天后过期或永不过期

### 7.4 移动端智能分享解析

**功能说明：**
在手机上从其他App（抖音、小红书、快手等）分享内容到链藏时，系统自动解析分享文本中的链接和标题。

**支持来源：**
- 抖音/快手/B站等短视频平台
- 小红书图文笔记
- 淘宝/京东/拼多多商品链接
- 微博/知乎内容分享
- 微信/公众号文章
- 网易云音乐等

**使用步骤：**
1. 在目标App中点击"分享"
2. 选择"复制链接"或"分享到LinkChest"
3. 在链藏中点击"+"添加收藏
4. 系统自动从剪贴板读取并解析
5. 自动提取链接、标题、平台信息

### 7.5 分享管理

**查看已创建的分享：**
1. 进入"分享管理"页面
2. 查看所有分享的列表
3. 显示分享标题、访问次数、创建时间

**停用分享：**
1. 在分享列表中找到目标分享
2. 点击"停用"按钮
3. 确认停用操作
4. 停用后分享链接立即失效

**复制分享链接：**
1. 在分享列表中点击"复制链接"
2. 链接复制到剪贴板
3. 可通过微信、邮件等方式发送

## 10. 常见问题

### 10.1 登录问题

**Q: 忘记密码怎么办？**
A: 在登录页面点击"忘记密码"，输入注册邮箱，系统会发送重置密码链接到您的邮箱。

**Q: 账号被锁定怎么办？**
A: 连续5次密码输入错误会导致账号锁定30分钟。请等待30分钟后重试，或联系客服解锁。

**Q: 收不到验证邮件？**
A: 请检查垃圾邮件箱，或确认输入的邮箱地址是否正确。如仍有问题请联系客服。

### 10.2 收藏问题

**Q: 收藏的链接打不开？**
A: 链接可能已失效或被删除。您可以编辑该收藏尝试更新链接，或删除该收藏。

**Q: 如何批量删除收藏？**
A: 进入收藏列表页面，点击"编辑"进入批量编辑模式，勾选要删除的收藏，点击底部"删除"按钮。

**Q: 封面图不显示？**
A: 部分网站可能禁止抓取封面图。您可以手动上传自定义封面替代。

**Q: 重复链接如何处理？**
A: 系统会自动检测重复链接。您可以在"重复检测"页面查看并处理重复项。

### 10.3 同步问题

**Q: 手机和电脑数据不同步？**
A: 数据实时同步，请确保两台设备都登录了同一账号且网络连接正常。下拉刷新可强制更新。

**Q: 离线时能使用吗？**
A: 部分功能需要网络连接。已加载的数据可在本地缓存中查看，但无法进行添加、删除等操作。

### 10.4 分享问题

**Q: 分享链接被提示密码错误？**
A: 该分享设置了访问密码，请联系分享创建者获取正确密码。

**Q: 分享链接提示已过期？**
A: 该分享已超过设置的过期时间。请联系分享创建者重新创建分享。

**Q: 能分享给没有链藏账号的人吗？**
A: 可以。公开分享链接无需登录即可查看，私有分享只需输入密码即可访问。

### 10.5 数据安全

**Q: 我的数据安全吗？**
A: 所有数据加密传输（HTTPS），密码采用bcryptjs加密存储，数据库定期备份。

**Q: 如何备份我的收藏？**
A: 可在"导出"页面将收藏导出为HTML或JSON格式保存到本地。也可开启"自动备份"功能定期备份。

**Q: 删除的收藏能恢复吗？**
A: 收藏被移动到回收站，可在回收站中查看和恢复。清空回收站后无法恢复。

### 11.1 系统架构

LinkChest链藏采用前后端分离的微服务架构：

| 层级 | 技术选型 | 说明 |
|------|----------|------|
| Web前端 | Next.js 14 + React 18 + Tailwind CSS | 桌面端/平板网页应用 |
| Mobile前端 | React Native + Expo SDK 52 | iOS/Android原生应用 |
| API服务 | Node.js + Express.js + TypeScript | RESTful API服务 |
| 数据库 | PostgreSQL 15 | 关系型数据库（主库） |
| 缓存 | Redis 7 | 高频数据缓存 |
| 认证 | JWT + bcryptjs | Token认证机制 |
| 部署 | Nginx + PM2 | 反向代理 + 进程管理 |
