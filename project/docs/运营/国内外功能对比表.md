## 十二、部署流程对比

| 步骤 | 国内版 | 海外版 |
|------|--------|--------|
| 拉取代码 | `cd /opt/linkchest/api && git pull` | `cd /opt/linkchest/api && git pull` |
| 构建API | `cd apps/api && npm run build` | `cd apps/api && npm run build` |
| 构建Web | `cd apps/web && npm run build:raw` | `cd apps/web && npm run build:raw` |
| 重启API | `pm2 restart linkchest-api` | `pm2 restart linkchest-api-global` |
| 重启Web | `pm2 restart linkchest-web` | `pm2 restart linkchest-web-global` |
