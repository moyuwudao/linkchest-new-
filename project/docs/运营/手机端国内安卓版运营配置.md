## 七、构建配置

### 7.1 EAS Build配置

```json
{
  "build": {
    "production-china": {
      "autoIncrement": true,
      "env": {
        "MARKET": "china"
      },
      "ios": {
        "podsDirectory": "ios/Pods"
      }
    }
  }
}
```

### 7.2 构建命令

```bash
# 使用EAS Cloud构建
eas build --profile production-china --platform android

# 本地构建
eas build --profile production-china --platform android --local
```

### 7.3 app.config.js配置

```javascript
{
  expo: {
    name: '链藏',
    android: {
      package: 'cn.linkchest.app',
      usesCleartextTraffic: true, // 国内版允许HTTP明文
      googleServicesFile: './google-services.json',
    },
    extra: {
      market: 'china',
    },
  }
}
```
