## 七、构建配置

### 7.1 EAS Build配置

```json
{
  "build": {
    "production-china": {
      "autoIncrement": true,
      "env": {
        "MARKET": "china"
      },
      "ios": {
        "podsDirectory": "ios/Pods"
      }
    }
  }
}
```

### 7.2 构建命令

```bash
# 使用EAS Cloud构建
eas build --profile production-china --platform android

# 本地构建
eas build --profile production-china --platform android --local
```

### 7.3 app.config.js配置

```javascript
{
  expo: {
    name: '链藏',
    android: {
      package: 'cn.linkchest.app',
      // usesCleartextTraffic: false, // 国内服务器已启用HTTPS，无需明文
    },
    extra: {
      market: 'china',
    },
  }
}
```

## 十、权限配置

国内Android版权限说明：

| 权限 | 状态 | 说明 |
|------|------|------|
| Internet | ✅ 需要 | 网络访问 |
| Camera | ✅ 需要 | 拍照上传图片 |
| Storage/Photos | ⚠️ 部分支持 | 仅支持上传图片，**不支持**保存图片到本地 |
| 微信SDK所需权限 | ✅ 需要 | 微信登录和分享 |

**功能限制**：
- ❌ **不支持扫码分享**：国内安卓版不支持公众号二维码扫描分享功能
- ❌ **不支持保存图片**：无法将内容保存/下载到本地相册
- ✅ **仅支持上传图片**：可以从相册选择图片上传，或通过相机拍照上传

## 十一、Intent Filters

国内安卓版分享功能说明：

**不支持的功能**：
- ❌ 扫码分享（二维码扫描）
- ❌ 保存图片到本地

**支持的功能**：
- ✅ 上传图片（从相册选择或拍照）
- ✅ 基础分享链接到链藏（SEND intent）：

```json
{
  "action": "SEND",
  "category": ["DEFAULT"],
  "mimeType": "text/plain"
}
```
