## 七、构建配置

### 7.1 EAS Build配置

```json
{
  "build": {
    "production-global": {
      "autoIncrement": true,
      "env": {
        "MARKET": "global"
      },
      "ios": {
        "podsDirectory": "ios/Pods"
      }
    }
  }
}
```

### 7.2 构建命令

```bash
# 使用EAS Cloud构建
eas build --profile production-global --platform android

# 本地构建
eas build --profile production-global --platform android --local
```

### 7.3 app.config.js配置

```javascript
{
  expo: {
    name: 'LinkChest',
    android: {
      package: 'com.linkchest.app',
      usesCleartextTraffic: false, // 海外版禁止HTTP明文
      googleServicesFile: './google-services.json',
    },
    extra: {
      market: 'global',
    },
  }
}
```

## 十、Intent Filters

海外版支持SEND intent用于分享链接到LinkChest：

```json
{

  "category": ["DEFAULT"],
  "mimeType": "text/plain"
}
```
